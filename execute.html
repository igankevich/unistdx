Execute child process with arguments and environment (WIP)

Use the following code to execute errno command to display error message in Russian locale.

sys::argstream args;
args.append("/bin/errno");
args.append(42);
sys::argstream env;
env.append("LANG=ru_RU.UTF-8");
sys::this_process::execute(args, env);

The above code is equivalent to running 

env LANG=ru_RU.UTF-8 errno 42

command in shell.
